I - Introduction :

this collage course focuses more on the Computer Science experience rather then tunnel visioning on networks purely.

1) Network Definition :

literary speaking a network can be many things; a computer network ⇒ (internet, bluetooth, wifi), a network of social connections ⇒ Social Media ,a electric network, a telephone network ⇒ GSM,etc...
any net shaped structure for sharing resources (electricity, date, water, gas) is in fact a network.

in that case a Computer Network is a ensemble of interconnected nodes which have connecting liaisons that allow the transfer of objects (data)

2) Nodes :

the building block of networks, (in the context of computing) a Node is a receiver, server or relay-er of information.
