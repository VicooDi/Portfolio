I - Definition :

Quaternions are a system composed of 4 numbers, 3 of them being imaginary numbers that represent an axis and one real number representing a angle.
this system is usually used in 3D space for rotation calculation specifically to solve an issue know as gimble lock(while plane imaginary numbers are used in 2D space for rotation)

I - µThe Euler System :

The Euler system is composed of 3 numbers (a matrix is more accurate than a vector) all real used to represent rotation in a 3D or 2D space.
for Example :


euler has 3 axis that can be rotated upon individually to form any angle.
this rotation is sometimes heavily effected by the order of application and in some cases the final results may even differ to solve this issue a new technique was created within the same system, Gimbals which basically consists of parenting the axis to each other (usually XYZ)

with this specific setup
X axis is unaffected by any other rotation
Y is affected by X
and Z is affected by any rotation .

although a problem can still arise from this..

  • Gimbal Lock :
    a Gimbal lock is what happens when upon rotation 2 Gimbals control the same axis.
    effectively this restricts movement in one axis making Euler a system that is not effective in 100% of all situations.
