I - Introduction :

Every data type has a corresponding pointer type EX : int ⇒ *int
such special data type is used to store the addressee of a variable stored in memory. aka a pre-declared variable

II - Concept :

Pointers store the addressee of data.
Essentially pointers deal with the storing of data within the RAM.
Referenced data has variable size depending on the type and content of said data.
The RAM is divided into blocks,each block is worth one bit and has an addressee, data can be stored in 1 or more blocks depending on it's size and type.

III - Application :

In the context of C
list of variables associated with pointers :

  • * : pointer's or de-pointer's a variable
  • malloc(int size) : allocates size amount of bits in the memory to a returned pointer. 
    char[] * ptr = (char*) malloc ( sizeof(char) * nbr_of_char);
  • free(* pointer) : free's up / deletes a pointer
  char[] * ptr = malloc ( sizeof(char) * nbr_of_char);
